## Account Setup

To get started with Dmitry Nesterenko Documentation, first create your account and set up your initial workspace. The platform offers a free tier for individual users and teams.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit the platform website and click the sign-up button. Provide your email address and create a secure password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your email for a verification link and click it to activate your account.
  </Step>
  <Step title="Create Workspace" icon="folder-plus">
    Set up your first workspace by providing a name and description for your documentation projects.
  </Step>
  <Step title="Invite Team" icon="users">
    Add team members by entering their email addresses and assigning appropriate permissions.
  </Step>
</Steps>

## API Integration

Integrate Dmitry Nesterenko Documentation with your development workflow using our REST API.

<CodeGroup tabs="curl,JavaScript">
  ```bash
curl -X POST https://api.docs.example.com/v1/projects \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "New Project",
    "description": "Project description",
    "visibility": "private"
  }'
  ```
  ```javascript
const createProject = async (token, projectData) => {
  const response = await fetch('https://api.docs.example.com/v1/projects', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(projectData)
  });
  
  if (!response.ok) {
    throw new Error(`HTTP error! status: ${response.status}`);
  }
  
  return await response.json();
};

// Usage
const project = await createProject('YOUR_API_TOKEN', {
  name: 'New Project',
  description: 'Project description',
  visibility: 'private'
});
  ```
</CodeGroup>

<Callout kind="tip">
  Generate an API token from your account settings to authenticate API requests securely.
</Callout>
