## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>
## New Features

- Added support for custom domains with SSL certificates
- Introduced advanced search with filters and faceting
- Implemented real-time collaboration features for team editing
- Added export functionality for multiple formats (PDF, HTML, JSON)

## Improvements

- Enhanced API performance with caching layer
- Improved user interface responsiveness on mobile devices
- Updated webhook payload structure for better consistency
- Optimized database queries for faster page loads
</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>
## New Features

- Complete redesign of the user interface with modern components
- Added support for OpenAPI 3.0 specification generation
- Implemented granular permission system for projects
- Introduced workspace-level settings and configurations

## Breaking Changes

- API endpoints now require authentication tokens
- Deprecated legacy webhook formats (use new payload structure)
- Removed support for old file formats (upgrade to Markdown)
- Changed default project visibility to private
</Update>

<Update label="2024-08-10" description="v1.2.0" tags={["feature", "bugfix"]}>
## New Features

- Added version control integration with Git repositories
- Implemented document templates for quick content creation
- Added bulk import functionality for existing documentation
- Introduced API rate limiting and usage monitoring

## Bug Fixes

- Fixed issue with image uploads in Safari browser
- Resolved webhook delivery failures for large payloads
- Corrected permission checks for project collaborators
- Fixed search indexing delays for newly created documents
</Update>
